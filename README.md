# README

### Installation:

#### 1. Create `.env` file in project directory and add (HMAC_SECRET can be `SecureRandom.hex`):
```
OPEN_WEATHER_API_KEY=xxxxxxxxxxxxxxxxxxxxxxxx
HMAC_SECRET=some_secret
```

#### 2. Build an docker image and run containers
```
docker-compose up
```
